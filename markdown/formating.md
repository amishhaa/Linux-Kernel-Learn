Markdown files also support basic html code. 

Number of hashes at the start of the sentence guives headings like <br>
like h1, h2, h3 in html.

`# Heading 1`
# Heading1
`## Heading2`
## Heading2
`### Heading 3`
### Heading 3
`#### Heading 4`
#### Heading 4

Lines can be broken with either two spaces or `<br>`.  
Use backticks to write html code and not execute it in a md file.  

we can use img tag as in html to add pictures.  
`<img src="assets/warning.png">`  
<img src="assets/warning.png">
